# 5e Inspired Character Sheet ESDLA 5e

Character sheet created for Roll20 inspired by ESDLA 5e roleplaying style.

## Features

- Dark Fantasy inspired interface
- Attribute and skill rolls
- Advantage / normal / disadvantage roll system
- Attack, inventory and effects management
- Automatic calculations (AC, initiative, modifiers)
- Exhaustion system with penalties
- Custom roll template
- Visual themes (Human, Elf, Dwarf, Dark)

## Author

David Sainz

## Instructions

1. Select the character sheet when creating a Roll20 game.
2. Fill in attributes and skills.
3. Use the roll buttons to perform checks and attacks.
4. Apply effects and exhaustion to modify rolls automatically.
