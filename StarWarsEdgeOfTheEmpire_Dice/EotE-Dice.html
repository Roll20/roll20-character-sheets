Adding Dice

To add additional dice write the number of dice and color letter you want to roll.

Example: 2b outputs

Example: 1b 1g outputs

b -> Blue = Boost
g -> Green = Ability
y -> Yellow = Proficiency
blk -> BLack = Setback
p -> Purple = Difficulty
r -> Red = Challenge
w -> White = Force
