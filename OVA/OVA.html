In order to get the Combat Stats roll buttons to work for existing characters with existing Stats, each stat will need to be changed so the sheet worker gets triggered. To accomplish this, you will need to change the number in each of the Roll boxes to a different number and hit enter, then you'll need to change it back and hit enter a second time. After that, each roll button should work. You should not need to do this when creating a new a character, nor when adding a new Combat Stat.

To hide this message, simply click on the downward pointing triangle above this message.
