Settings

Player's Gender Pronoun
In this area you can denote your character's correct gender pronoun. This will change your pronoun in rolls. The first box is for the character's determiner. The second is for the character's pronoun. The third is for the character's matching conjugation of have.
Persona's Gender Pronoun
In this area you can denote your persona's correct gender pronoun. This will change your persona's determiner in rolls.
Testing Mode
In this area you can set the name that is show when moves are rolled out. If set to "Default" your character and persona's correct name will be shown in rolls. If set to "Testing" "???" will be displayed instead (hiding the character sheet associated with the roll). The first box is for the Persona User, the second is for the Persona.
Attribute Help
In this area there are some attributes to help you code macros. The first shows what to write in Dia. The second shows Dia with LV/4 (as with Temperance's Divine Grave). The third shows Life Drain being used by a non-Death character. To have a stat automatically pulled type in @{Xcalc} where X is the stat. To have a stat round up type in ceil(X/Y). To roll (Willpower)D6, all you have to do is put the LV/4 in parenthesis before D6.
+4 Rule
In this area you can choose whether or not you wish the +4 rule to apply to you. If you select "+4 Cap" the max you can add to a roll is 4. If you select "No Cap" bonuses to rolls are unrestricted.
