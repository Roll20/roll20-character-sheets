# Gangbusters BX Character Sheet

Roll20 character sheet for **Gangbusters BX Edition** by Mark Hunt (Chubby Funster / Wizard Tower Publishing).

## Tabs

- **Character** — Matches the official 1-page record sheet: identity, AC/HP, abilities, saves (Moxie, Quickness, Toughness, Driving, Observation), languages, special skills, to-hit chart, and weapons.
- **Equipment & Notes** — Character background, inventory, money ($/¢), vehicle, session notes, and other storage.

## Features

- Class autofill for all 8 classes (levels 1–14): title, HD, XP to next level, saving throws
- Standard Combat Chart to-hit values (levels 1–3 vs 4+)
- B/X ability modifiers with d20 roll buttons
- Save rolls (d20 ≥ target) with Moxie/Wisdom, Quickness/Dexterity, Toughness/Constitution
- Active weapon selection for the Dmg button
- Roll templates: `gbx_save`, `gbx_attack`, `gbx_check`, `gbx_roll`
