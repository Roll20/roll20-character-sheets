## Changelog

### V.1.3 (2026-06)
* Two tabs: **Character** and **Mount & Equipment** (removed redundant Equipment tab)
* Session notes on Character tab; inventory split across Personal, Mount, and Other sections
* Active weapon selection drives the Damage button roll
* Experience and Combat sub-panels; bold AC and Level values; improved roll-button styling

### V.1.2 (2026-06)
* Class autofill: title, HD, XP-to-next-level, saving throws, and to-hit row (all 11 PC classes)
* Cowhand uses Singing Cowboy saves; Doctor has no class table (manual entry)

### V.1.1 (2026-06)
* Layout alignment fixes: identity fields, abilities, saves, to-hit row, weapons grid
* Added Lawman, Peddler, and Cowhand to class dropdown (from supplemental PDFs)

* **Peddler_Character_Class_...pdf** — third-party Peddler class (OL/BR/AP/LD/SP skills)

### V.1.0 (2026-06)
* Initial release for Tall Tales (Wild West BX)
* Character and Mount & Equipment tabs based on the official BX record sheets
* Six abilities with auto-calculated BX modifiers
* Tall Tales saving throws (Gumption, Quickness, Toughness, Riding, Observation)
* Melee/missile attack rolls, saving throws, and ability checks
* AC 9–0 "to hit" reference row and repeating weapons list
