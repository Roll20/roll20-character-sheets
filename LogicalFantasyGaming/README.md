# Logical Fantasy Gaming (LFG)

**System:** Logical Fantasy Gaming (LFG)  
**Author:** Daniel Brillat  

## Description
Logical Fantasy Gaming (LFG) is a proprietary tabletop role-playing game system focused on
structured, rules-driven character progression, tactical combat, and clear mechanical
resolution.

This Roll20 character sheet supports core LFG gameplay and is designed for use with the
official LFG ruleset.

## Sheet Features
- Supports Melee, Hybrid, and Caster characters
- Attribute and derived stat tracking
- Skill ranks, bonuses, and totals
- Equipment, armor, saves, and encumbrance tracking
- Custom layout tailored specifically to LFG mechanics
