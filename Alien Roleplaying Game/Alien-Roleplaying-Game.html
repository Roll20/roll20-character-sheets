v3.00
(2021-12-28)
Added configuration tab for sheet level options and customizations, like input fields and background image.
Added support for API rolls using the Alien RPG dice roller script. This requires a Pro account.
Added default sheet settings for all the configuration options.
Added weights and encumbrance calculation as an option. Also added a repeating section at the end of teh gear list, to allow adding 9 items or more.
Added option to calculate Health automatcially, and included a Tough checkbox to include in calculation.
Added indicators for tabs of lists making it easier to see where there is data in the tabbed list.
Thanks Richard M for this!
Added weights and encumbrance calculation as an option. Also added a repeating section at the end of teh gear list, to allow adding 9 items or more.
Added a repeating section at end of Gear list to allow for exapanding it, and made the panel scrollable. Also added an option to only use the repeating section, and hide the static gear rowws.
Added four more talent slots.
Added option for squarer input boxes and option to enable spinner (up and down buttons) for number fields.

Fixed counting of successes also for Stress dice.
