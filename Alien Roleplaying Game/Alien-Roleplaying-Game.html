Change log
v5.00
(2022-07-09)
Major version update but no changes to used attributes. Implemented the Char Sheet Enhancement (CSE) and thus moved away from Legacy styling. As part of this, a lot of improvements in the styling was done, e.g. implementing label "for" tag attributes for inputs.
Added options to include specific rolls for talents, based on Attribute and Skill with modifier or for specific Consumable supply roll. The latter is meant to cover talents like Light Eater by selecting Food and -2 modifier.
Added names of talents, weapons and armament to their respective tab to make navigating easier.
Also added more Armament tabs to cover for more powerful ships.
Added an option to include more Permanent skill modifiers, to cover bonuses or penalties from gear.
Also added a button to disable the Modifier Query popup for rolls, the popup can get lost if using the sheet popped out.
Added handling of Permanent Rads through a dropdown underneath the radiation boxes, as well as rolls for Radiation damage and Radiation recovery.
Added an option Token Initiative to remove the attempts to add a token initiative to turn tracker, thus avoiding any unnecessary and annoying error messages in chat. The button is above and to the right of the Roll buttons section.
Added an option for Nerves of Steel talent. Please note that specific talent modifications like this may later on be moved to the talent slots.
Made sure there is a Secret Rolls and Modifier Query option on each tab, for easy access.
Tested upgrade from previous legacy styling, fixed identified issue with macro quick bar rolls. Roll names now are prefixed with "sheet-" to provide consistency with previous version, the prefix seemed to be added automatically by the system in legacy styling. It is then removed for macros but leaving the "-" still as prefix (PM for questions or issues).
Made the Randomize attack button for Hostiles work with the quick macro bar.
Please note this means the roll button name is now "sheet-randomizeattack" and the action button is "randomizeattackaction", for abilities to use "%{selected|randomizeattackaction}" for token action.
v4.04
(2022-05-04)
Fixed roll commands error caused by attribute for handling secret rolls not being initiatilised.
Added a randomzier button, individual roll buttons and additional options for Hostile's Signature attacks. The options include D6 dice, Base attack dice and Base damage.
Added a "Copy above" button, [ | ] , in Signature attacks to make it easier to configure same outcome on a second roll result, e.g. "The hostile will attack with 6 dice on roll 1-2", can be achieved by using the Copy button on Signature attack 2.
Renamed Damage to Base Damage in Rolltemplate, and moved the roll information Bonus, Base damage and Range to above the dice output. This hopefully makes it clearer what is the static information for the roll and the resulting output, i.e. Successes and Panic.
v4.03
(2022-04-05)
Fixed spelling mistake in API roll for Weapon 5.
v4.02
(2022-03-30)
Fixed the handling of secret (whispered) rolls so that it works across tabs and different roll types. Also, it no longer resets when opening the sheet. An upgrade function will ensure setting stays same as before.
Please note an upcoming release will fix the Default Sheet Settings, and it seems like there is no way to prevent the game default settings from overriding the local settings for each sheet. This change will be announced later, and will require you to reset your sheet configuraiton if different from the game default.
v4.01
(2022-03-20)
Mobility and Observability macros now work the same for Character and Hostile tab, same macro should be usable for either. Technically, they both use the attribute and the skill, but for the Hostile the attribute would be kept at the default 0 level.
The Roll template Success and Panic calculations are now fixed from the previous version, it kept showing 0 no matter how many successes or facehuggers one rolled.
The Roll template chat output now states Panic for the facehugger results, to make similar to the API Script output.
Selecting secret rolls when using the API Script now honors the setting for rolls using the Roll template.
v4.00
(2022-03-09)
Added the Hostile tab to use for Xenos and other nasty things that lurk in the darkness of space.
Adjusted the size of the sheet frame to remove white border in dark mode. Space is a very dark place.

{{/rollTotal() panic-roll 8}} {{#rollTotal() panic-roll 9}}
DROP ITEM. Whether by stress, confusion or the realization that you're all going to die anyway, you drop a weapon or other important itemthe GM decides which one. Your STRESS LEVEL increases by one.
{{/rollTotal() panic-roll 9}} {{#rollTotal() panic-roll 10}}
FREEZE. You're frozen by fear or stress for one Round, losing your next slow action. Your STRESS LEVEL, and the STRESS LEVEL of all friendly PCs in SHORT range of you, increases by one.
{{/rollTotal() panic-roll 10}} {{#rollTotal() panic-roll 11}}
SEEK COVER. You must use your next action to move away from danger and find a safe spot if possible. You are allowed to make a retreat roll (see page 93) if you have an enemy at ENGAGED range. Your STRESS LEVEL is decreased by one, but the STRESS LEVEL of all friendly PCs in SHORT range increases by one. After one Round, you can act normally.
{{/rollTotal() panic-roll 11}} {{#rollTotal() panic-roll 12}}
SCREAM. You scream your lungs out for one Round, losing your next slow action. Your STRESS LEVEL is decreased by one, but every friendly character who hears your scream must make an immediate Panic Roll.
{{/rollTotal() panic-roll 12}} {{#rollTotal() panic-roll 13}}
FLEE. You just can't take it anymore. You must flee to a safe place and refuse to leave it. You won't attack anyone and won't attempt anything dangerous. You are not allowed to make a retreat roll (see page 93) if you have an enemy at ENGAGED range when you flee. Your STRESS LEVEL is decreased by one, but every friendly character who sees you run must make an immediate Panic Roll.
{{/rollTotal() panic-roll 13}} {{#rollTotal() panic-roll 14}}
BERSERK. You must immediately attack the nearest person or creature, friendly or not. You won't stop until you or the target is Broken. Every friendly character who witnesses your rampage must make an immediate Panic Roll.
{{/rollTotal() panic-roll 14}} {{#rollGreater() panic-roll 14}}
CATATONIC. You collapse to the floor and can't talk or move, staring blankly into oblivion.
