# Cerbère

Une fiche Roll20 pour le JdR Cerbère de John Grümph, un système presque générique pour jeux de rôle et d’aventure sérieuse.

Le jeu est disponible aux [éditions Chibi](https://legrumph.org/Terrier/public/chibi/cerbere).

Fiche maintenue par [@Choco_Lanoar](https://github.com/ChocoLanoar).

# Suivi de versions

## Version 1.0

Version initiale. Fonctionnalités de base :
- Mode édition / mode vue.
- Jets de dés standards et jets de dégâts / soins, à 1, 2 ou 3 d10
- Suivi des PV / PC / PR / XP
- Ajout libre de compétences / cultures / maîtrises / talents
- Ajout d’équipement

Fonctionnalités non-implémentées :
- Relations
- Motivations et contraintes
